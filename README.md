# Titanic-survival-analysis

## Project Overview
This project involves a comprehensive data cleaning and analysis of the Titanic dataset. The goal was to transform raw, messy passenger data into meaningful insights using Python.

## Key Steps Performed:
* **Data Cleaning:** Handled missing values (imputed using median/mode) and removed duplicate records.
* **Feature Engineering:** Analyzed key factors affecting passenger survival.
* **Data Visualization:** Used Matplotlib to generate insights, including:
    - Age distribution of passengers.
    - Survival rate comparison by Gender.
    - Survival rate comparison by Passenger Class.

## Tools Used:
* Python
* Pandas
* NumPy
* Matplotlib

## How to view the project:
You can view the full analysis in the `Titanic_Analysis.ipynb` file.
